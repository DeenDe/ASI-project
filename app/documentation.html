Documentation

This website is a personal project, built by me using HTML, CSS, and a bit of JavaScript. It is hosted on a Kubernetes cluster, using a container orchestration platform called Kubernetes.

The website is powered by a server side script written in Python. The script is quite simple - it just counts the number of unique visitors to the website, and stores that data in a file. The script uses cookies to track whether a user has been to the website before or not. If the user has not been to the website before, the script will increment the number of unique visitors. The script is deployed on a Kubernetes cluster, using a tool called ArgoCD. ArgoCD is a continuous deployment tool for Kubernetes, which automates the deployment of applications to Kubernetes.

The website is also hosted on a container orchestration platform called Kubernetes. Kubernetes is a popular container orchestration platform, which allows you to easily deploy, manage, and scale containerized applications. The website is containerized as a Docker container, and is deployed to Kubernetes using a tool called Helm. Helm is a package manager for Kubernetes, which allows you to easily install, upgrade, and manage applications on Kubernetes.
