Documentation

This website was built using HTML, CSS, and a bit of JavaScript. Server-side sripts and cookies was also used. Webside is running with node.js, version v20.12.2. The webside is containerized and hosted on Kubernetes platform.

The website implements a unique visit tracking system utilizing cookies and server-side scripts. Each visitor is identified uniquely, and their visit is recorded to ensure accurate statistics. The visit count is stored in a file on the server, maintaining a record of unique visits.

Efforts have been made to ensure clear navigation throughout the website. The navigation menu is prominently displayed, making it easy for users to explore different sections. The current page is highlighted in the navigation bar, aiding users in understanding their location within the site.

A highlight of the website is the technology gallery, showcasing various technologies with their logos and descriptive text. Users can hover over each technology to view additional information, providing a comprehensive overview of the featured technologies. All graphics have also text description.

The website includes a brief description of the author, offering visitors insight into the creator behind the content. This personal touch adds authenticity and fosters a connection between the author and the audience.
